# F1 Race Tracker API

A RESTful API built with Spring Boot 4 and PostgreSQL for managing
Formula 1 drivers, teams, races, and race results.

## Tech Stack
- Java 21
- Spring Boot 4.0.5
- Spring Data JPA
- PostgreSQL
- Maven
- Lombok

## Endpoints
- GET /api/drivers
- GET /api/races/{year}
- POST /api/results
- GET /api/standings/{year}
